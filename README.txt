Instrucciones para ejecutar:
Se necesitan los archivos postfix.py y postfixLog.py en la misma carpeta de yahooo.py para ejecutar el programa.
Me he dado cuenta que si se abre el archivo yahooo.py con VisualStudio Code no encuentra el archivo yahooo.txt a menos de que se abrea la carpeta completa del proyecto,
espero que no pase con otros editores de texto.
Consideraciones:
Como se dijo en el video, se considera que el usuario no va a agregar más de 4 variables
Se considera que al asignarle un valor a una variable, el usuario está ingresando el nombre de una variable creada anteriormente.
El resultado de las operaciones es el valor redondeado.
Se considera como main correcto el primer Start Game y Game Over que aparezca.
Se considera que al llamar una función, si el parametro es un número, este sera solo positivo, para numeros negativos introducir una variable con valor negativo.
Se considera que cuando el usuario llama a una función en el main, esta ya fue definida.
